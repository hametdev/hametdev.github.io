FAQ

Common questions about web development

Koliko košta izrada web stranice? expand_more
Jednostavna responzivna web stranica s SEO-om počinje od 190 €. Web stranice s više podstranica i custom dizajnom od 350 €. Web aplikacije s korisničkim računima i bazom podataka cijenimo individualno. Procjena je uvijek besplatna.
Koliko traje izrada web stranice? expand_more
Jednostavnu web stranicu ili landing page isporučujemo za 3–7 dana. Složenije stranice s više podstranica i custom funkcionalnostima — 2–4 tjedna. Web aplikacije cijenimo i planiramo individualno.
Hoće li moja web stranica biti SEO optimizirana? expand_more
Da. Svaka web stranica dolazi s tehničkim SEO-om: optimizirane meta oznake, brzina 90+ PageSpeed, strukturirani podaci i mobilna prilagodba. Nudimo i napredni SEO audit za postojeće stranice.
Mogu li sami uređivati sadržaj web stranice? expand_more
Da. Implementiramo CMS (sustav za upravljanje sadržajem) koji vam omogućuje samostalno uređivanje tekstova, slika i blog objava — bez ikakvog znanja programiranja.
Nudite li hosting i tehničku podršku? expand_more
Da. Možemo preuzeti kompletno upravljanje hostingom, SSL certifikatima i periodičnim nadogradnjama. Pružamo i tehničku podršku nakon lansiranja — brzo reagiramo na sve probleme.
